# fraud-detection-ml

ML model to detect fraudulent bank transactions - Portfolio Project



\## 📂 Dataset



Les données utilisées dans ce projet proviennent de Kaggle :



🔗 \[Credit Card Fraud Detection - Kaggle](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)



\### Comment télécharger le dataset ?



1\. Crée un compte gratuit sur \[kaggle.com](https://www.kaggle.com)

2\. Clique sur le lien ci-dessus

3\. Clique sur \*\*Download\*\* pour télécharger `creditcard.csv`

4\. Place le fichier dans le dossier `data/` du projet



> ⚠️ Le fichier CSV n'est pas inclus dans ce repo (trop volumineux).

> Il doit être téléchargé séparément depuis Kaggle.
